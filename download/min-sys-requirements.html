    Abstractverse Minimum System Requirements (version v1.0.0a):

      All Platforms

      At least 512 MB of memory.
      At least 10 MB of free space.
      (NVIDIA) Geforce GT 605 or later
      (AMD) Radeon HD 7750 or later
      (INTEL) Intel HD Graphics 4000 or later
      (If your GPU isn't on here, check here)
      OR view the advanced MSR info.

      Windows

      OS: Windows 7 or later (with Vulkan 1.0 driver support.)

      Linux

      Run the command "vulkaninfo" in your terminal if you have vulkan-tools installed.
      If you don't want to do that, check the advanced MSR info.

      MacOS & iOS

      MacOS: MacOS 10.11 or later (with MoltenVK API support.)
      iOS: iOS 9.0 or later (with MoltenVK API support.)

      Android

      OS: Android 7.0 or later (with Vulkan 1.0 API support.)
      (SAMSUNG) Samsung Galaxy S7 or later
      (GOOGLE) 1st gen Google Pixel or later
      Don't see your phone here? Try viewing the advanced MSR info.

      If you need more information other than the advanced MSR info, check https://vulkan.gpuinfo.org/
